# Z3

A simple CLI for scaffolding Baidu-wallet projects.

### Installation

Prerequisites: [Node.js](https://nodejs.org/en/) (>=4.x, 6.x preferred), npm version 3+ and [Git](https://git-scm.com/).

``` bash
$ npm install -g z3
```

### Usage

``` bash
$ z3 init <template-name>
```

Example:

``` bash
$ z3 init spa 
```
